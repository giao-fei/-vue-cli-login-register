# 利用vue-cli+mongodb+nodejs+axios+express登录注册小案例<br>
>1- 下载nodejs并配置好环境变量<br>
>2- 下载@vue/cli<br>
>>     npm install @vue/cli
>>     vue init created '项目名称'
>3- 下载mongodb<br>
>>     npm install mongodb
>4- 下载axios库<br>
>>     npm install axios
>5- 下载express框架<br>
>>     npm install express --save
>>     npm install express-body --save
